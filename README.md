## Control Instructions:

I've implemented 4 scenes in this code. 

*On/Off Light State turn the lamp on and off.*

on your Keyboard<br>
-press 'o' to turn the lamp On.<br>
-press 'f' to turn the lamp Off.

*Day/Night time*

-press 'd' for day time. (Default)<br>
-press 'n' for night time.
